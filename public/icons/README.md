# PWA Icons

Ce dossier contient les icônes pour la Progressive Web App (PWA).

## Tailles requises

### Icônes Android/Chrome
- 72x72px
- 96x96px
- 128x128px
- 144x144px
- 152x152px
- 192x192px (maskable)
- 384x384px
- 512x512px (maskable)

### Icônes iOS/Apple
- 120x120px (iPhone)
- 152x152px (iPad)
- 167x167px (iPad Pro)
- 180x180px (iPhone Plus)

## Génération

Utilisez Sharp ou un service en ligne comme https://realfavicongenerator.net/
pour générer toutes les tailles nécessaires à partir de l'image source.

## Format

- Format: PNG avec transparence
- Fond: Transparent ou blanc selon l'usage
- Qualité: Haute résolution pour éviter le flou
